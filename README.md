# lein-as-resource

Provides a dependency containing a lein uberjar. See [alembic](https://github.com/pallet/alembic) for why this might be useful.

## Usage

```clj
[lein-as-resource "2.7.1"]
```

The lein uberjar is provided as "lein-standalone.jar".
